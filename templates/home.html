What is PubFolder?

PubFolder makes it easy to host files to the public straight out or your Dropbox. It works just like Dropbox's discontinued Public folder.

How is this different than "Copy Dropbox Link" or "Sharing" in Dropbox?

Two ways.
  1. Dropbox share links embed your content within a Dropbox web UI. PubFolder gives you a link directly to your raw file. This is particularly useful in some situations, like if you're hosting an image which you'd like to embed somewhere.
  2. PubFolder links come in a predictable format. The path to each file is pubfolder.com/<your user id>/<the file path within PubFolder>

Are my PubFolder files safe?

Files in your PubFolder are NOT private. Putting a file in your PubFolder shares it with the world in a guessable link format. Besides that, PubFolder is very safe.
