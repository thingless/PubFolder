What is PubFolder?

PubFolder makes it easy to host files to the public straight out or your Dropbox. It works just like Dropbox's old Public folder.

How is this different than "Copy Dropbox Link" or "Sharing" in Dropbox?

Two ways.
1) Dropbox share links embed your content within a Dropbox web UI. PubFolder hosts your raw file directly. This is particularly useful in some situations, like if you're hosting an image which you'd like to embed somewhere.
2) PubFolder links come in a predictable format. The path to each file is pubfolder.com/<your user id>/<the file path within PubFolder>
